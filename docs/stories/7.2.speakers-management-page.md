## Story

**As an** organizer,
**I want** a dedicated speakers management page with email invitations, CSV import, pending/confirmed lists,
**so that** I can efficiently invite and manage all speakers for my event.

## Context

Atualmente palestrantes são gerenciados apenas via modal dentro do detalhe do evento (Story 4.2). A referência visual (`referencias/scribia-speakers.html`) define uma página standalone completa com:
- Breadcrumb de navegação
- Convite por email com tags visuais
- Import CSV + download modelo
- Lista de convites pendentes (com reenviar/expirado)
- Lista de confirmados com avatar, palestra e status

**Design System:** Seguir rigorosamente `docs/design-system.md`.

## Acceptance Criteria

1. Página `/dashboard/speakers` acessível pela sidebar (nav item "Palestrantes")
2. **Breadcrumb:** "TechSummit Brasil 2025 > Palestrantes"
3. **Page header:** título "Adicionar Palestrantes", subtítulo "Convide por e-mail ou importe uma lista CSV", badge "X confirmados · Y pendentes"
4. **Seção de convite por email:**
   - Input com ícone de envelope, placeholder "nome@empresa.com.br"
   - Botão "Enviar convite" (primary purple com ícone send e glow)
   - Área de tags: emails adicionados aparecem como chips removíveis (purple-dim, border-purple)
   - Ao pressionar Enter ou clicar "Enviar", email vira tag
5. **Import alternativo:**
   - Divider visual "— ou —"
   - Botão "Importar CSV" (ghost com ícone upload)
   - Botão "Baixar modelo" (ghost com ícone download)
6. **Lista de convites pendentes (card esquerdo):**
   - Header com título + badge de contagem
   - Tabela: E-mail | Enviado em (tempo relativo) | Status (Chip: Pendente/Expirado) | Ações
   - Ações: ícone reenviar (refresh), ícone remover (trash, danger hover)
   - Chip amarelo "Pendente", chip vermelho "Expirado" (>24h)
7. **Lista de palestrantes confirmados (card direito):**
   - Header com título + badge de contagem
   - Lista de itens: avatar (initials 36px) | nome + palestra vinculada | chip "Ativo" (green)
8. **Grid layout:** 2 colunas iguais para as listas
9. **Animação:** fadeUp nos cards de seção
10. Dados filtrados por evento ativo do organizador (RLS)
11. Sidebar nav item "Palestrantes" mostra estado ativo quando nesta página

  .from('speakers')
  .select('*, lectures!inner(title, event_id)')

**CSV Template:**
```csv

"João Silva","joao@empresa.com","TechCorp","CTO"
```

**Tempo relativo:**
```typescript
function timeAgo(date: string): string {
  const diff = Date.now() - new Date(date).getTime()
  const hours = Math.floor(diff / 3600000)
  if (hours < 1) return 'agora'
  if (hours < 24) return `há ${hours}h`
  const days = Math.floor(hours / 24)
  return `há ${days}d`
}
```
