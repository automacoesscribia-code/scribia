## Story

**As a** operator,
**I want** to select an audio input device and record continuously,
**so that** the lecture audio is captured in real-time with high quality.

## Acceptance Criteria

1. Listagem de dispositivos de áudio disponíveis via Tauri command `list_audio_devices()`
2. Seletor de dispositivo na UI com nome e tipo (microfone, line-in, etc.)
3. Tauri command `start_capture(device_id, lecture_id)` inicia gravação contínua
4. Áudio capturado via `cpal` (Rust) com sample rate 44.1kHz ou 48kHz
5. Encoding em tempo real para Opus/WebM para compressão eficiente
6. Buffer em memória de 30 segundos antes de salvar chunk em disco
7. Tauri command `stop_capture()` finaliza gravação e retorna metadata (duração, chunks gerados)
8. Tauri command `get_capture_status()` retorna: estado (idle/recording/paused), duração atual, nível de áudio (VU meter)
9. UI mostra: timer de duração, indicador de nível de áudio (VU meter), botão pause/resume/stop
10. Gravação sobrevive a micro-interrupções de áudio (buffer handling)

## Dev Notes

**Rust crate dependencies:**
```toml
cpal = "0.15"        # Audio capture
opus = "0.3"         # Opus encoding (or audiopus)
hound = "3.5"        # WAV writing (fallback)
webm = "0.1"         # WebM container (or custom)
```

**Capture architecture:**
```
Microphone → cpal stream → Ring Buffer (30s)
                              ↓
                          Opus Encoder → Chunk files (.webm)
                              ↓
                          Temp directory: {app_data}/captures/{lecture_id}/
```

**Tauri command signatures:**
```rust
#[tauri::command]
fn list_audio_devices() -> Result<Vec<AudioDevice>, String>

#[tauri::command]
fn start_capture(device_id: String, lecture_id: String) -> Result<(), String>

#[tauri::command]
fn stop_capture() -> Result<CaptureResult, String>

#[tauri::command]
fn get_capture_status() -> Result<CaptureStatus, String>

struct AudioDevice { id: String, name: String, sample_rates: Vec<u32> }
struct CaptureResult { duration_seconds: u64, chunks_count: u32, temp_dir: String }
struct CaptureStatus { state: String, duration_seconds: u64, audio_level: f32 }
```
