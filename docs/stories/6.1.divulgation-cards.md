## Story

**As an** organizer,
**I want** to generate social media cards for each lecture,
**so that** I can promote the event content on social media.

## Acceptance Criteria

1. Edge Function `_jobs/generate-card.ts` gera card de divulgação automaticamente
2. Card inclui: título da palestra, nome do palestrante, nome do evento, data, resumo curto (1-2 frases)
3. Card gerado como imagem PNG via `satori` (JSX→SVG) + `@resvg/resvg-js` (SVG→PNG via WASM) — Deno-native, sem dependências externas
4. Template visual com branding ScribIA (cores, logo, layout) definido como componente JSX
5. Imagem salva no Storage: `materials/{event_id}/{lecture_id}/card_og.png` e `card_square.png`
6. URL salva em `lectures.card_image_url`
7. No admin panel: preview do card + botão "Download" + botão "Regenerar"
8. Possibilidade de customizar texto do card antes de gerar

## Dev Notes

**Satori (JSX → SVG → PNG):**
```typescript
import satori from 'satori'
import { Resvg } from '@resvg/resvg-js'

const svg = await satori(
  <div style={{ /* card layout */ }}>
    <h1>{lecture.title}</h1>
    <p>{lecture.speaker.name}</p>
    <p>{lecture.summary.slice(0, 100)}...</p>
  </div>,
  { width: 1200, height: 630, fonts: [...] }
)
const png = new Resvg(svg).render().asPng()
```
