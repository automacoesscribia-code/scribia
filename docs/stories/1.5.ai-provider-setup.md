## Story

**As a** system,
**I want** a model-agnostic AI provider layer and shared PDF/DOCX generation utilities,
**so that** all processing Edge Functions can generate content and documents using any AI provider (Gemini default) without vendor lock-in, and share PDF generation logic.

## Acceptance Criteria

1. Package `@scribia/shared/ai` com interface `AIProvider` e factory `createAIProvider()`
2. `GeminiProvider` implementado usando `@google/generative-ai` (Gemini 2.5 Flash default)
3. `ClaudeProvider` implementado usando `@anthropic-ai/sdk` (fallback/upgrade)
4. Provider selecionado via env var `AI_PROVIDER` (default: `gemini`)
5. Método `generateContent(prompt, options)` retorna string (texto livre)
6. Método `generateStructured<T>(prompt, schema)` retorna JSON tipado (usando Zod schema)
7. Package `@scribia/shared/pdf` com classe `ScribiaPdfBuilder` usando `pdf-lib` (Deno-native)
8. `ScribiaPdfBuilder` suporta: `addCover()`, `addTableOfContents()`, `addChapter()`, `addFooter()`, `build()` → `Uint8Array`
9. Template com branding ScribIA: fonte profissional, cores (#1a1a2e, #e94560), logo embeddável
10. Package `@scribia/shared/docx` com helper `ScribiaDocxBuilder` usando `docx` (npm:docx compatível com Deno)
11. `ScribiaDocxBuilder` suporta: headings, paragraphs, lists, tables, page numbers, header/footer
12. Ambos builders exportados e reutilizáveis por Edge Functions (Stories 3.3, 4.5, 6.2)
13. Vitest: testes unitários para AI Provider (mock APIs) e PDF/DOCX builders (verificar output)

## Dev Notes

**AI Provider interface:**
```typescript
// packages/shared/src/ai/types.ts
import { ZodSchema } from 'zod'

export interface GenerateOptions {
  maxTokens?: number
  temperature?: number
  systemPrompt?: string
}

export interface AIProvider {
  generateContent(prompt: string, options?: GenerateOptions): Promise<string>
  generateStructured<T>(prompt: string, schema: ZodSchema<T>, options?: GenerateOptions): Promise<T>
}
```

**Factory:**
```typescript
// packages/shared/src/ai/factory.ts
export function createAIProvider(): AIProvider {
  const provider = process.env.AI_PROVIDER ?? Deno.env.get('AI_PROVIDER') ?? 'gemini'
  switch (provider) {
    case 'gemini': return new GeminiProvider()
    case 'claude': return new ClaudeProvider()
    default: throw new Error(`Unknown AI provider: ${provider}`)
  }
}
```
