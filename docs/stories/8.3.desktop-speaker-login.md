## Story

**As a** speaker,
**I want** to log into the desktop app and see only my assigned lectures,
**so that** I can select the correct lecture to record.

## Context

O desktop app atualmente mostra TODOS os eventos e palestras do sistema apos login. Para speakers, o app deve:

1. Detectar que o usuario logado tem role `speaker`
2. Buscar o registro `speakers` onde `user_id = auth.uid()`
3. Buscar `lectures` onde `speaker_id` = id do speaker
4. Mostrar apenas essas palestras, agrupadas por evento
5. Cada palestra mostra titulo, evento, data agendada e status

**Regra:** Super admin tambem pode logar no desktop e ver TODOS os eventos/palestras (funcionalidade existente mantida).

## Acceptance Criteria

1. Apos login, app detecta role do usuario (speaker vs organizer vs super_admin)
2. Se role=speaker: busca `speakers` WHERE `user_id = auth.uid()` para obter `speaker.id`
3. Se role=speaker: busca `lectures` WHERE `speaker_id = speaker.id` com dados do evento (JOIN events)
4. Tela mostra lista de palestras do speaker com: titulo, nome do evento, data agendada, status
5. Se nao ha palestras vinculadas: mensagem "Nenhuma palestra atribuida a voce ainda"
6. Se role=super_admin ou organizer: comportamento atual mantido (todos os eventos)
7. Speaker clica na palestra → vai para tela de gravacao (RecordingSession)
8. Header mostra nome do speaker e botao "Sair"
9. Card da palestra mostra indicador visual de status: "Agendada" (azul), "Gravada" (verde), "Em gravacao" (amarelo)
10. Ao ter audio local salvo (chunks na pasta temp), card mostra badge "Audio salvo localmente"

## Dev Notes

**Query para palestras do speaker:**
```typescript
// 1. Buscar speaker record
const { data: speaker } = await supabase
  .from('speakers')
  .select('id')
  .eq('user_id', user.id)
  .single()

// 2. Buscar palestras vinculadas
const { data: lectures } = await supabase
  .from('lectures')
  .select('id, title, status, scheduled_at, events(id, name)')
  .eq('speaker_id', speaker.id)
  .order('scheduled_at', { ascending: true })
```

**Screen state machine update:**
```typescript
type Screen =
  | { type: 'login' }
  | { type: 'events' }                    // organizer/super_admin
  | { type: 'speaker-lectures' }          // speaker
  | { type: 'recording'; eventId: string; lectureId: string; ... }
```

**Tauri command:**
```rust
#[tauri::command]
fn check_local_chunks(lecture_id: String) -> Result<LocalChunkInfo, String> {
    let temp_dir = std::env::temp_dir().join("scribia").join(&lecture_id);
    if !temp_dir.exists() {
        return Ok(LocalChunkInfo { exists: false, chunk_count: 0, total_bytes: 0 });
    }
    let chunks: Vec<_> = std::fs::read_dir(&temp_dir)?
        .filter_map(|e| e.ok())
        .filter(|e| e.path().extension().map_or(false, |ext| ext == "wav"))
        .collect();
    let total_bytes: u64 = chunks.iter().map(|e| e.metadata().map(|m| m.len()).unwrap_or(0)).sum();
    Ok(LocalChunkInfo { exists: true, chunk_count: chunks.len() as u32, total_bytes })
}
```
