## Story

**As a** system,
**I want** to automatically transcribe audio files using OpenAI Whisper API,
**so that** lecture content is available as text for further processing.

## Acceptance Criteria

1. Edge Function `_jobs/transcribe.ts` processa jobs do tipo `transcription` da fila
2. Download do áudio final do Supabase Storage
3. Envio para OpenAI Whisper API com language hint `pt` (português)
4. Transcrição salva em `lectures.transcript_text`
5. Job status atualizado: `queued → processing → completed/failed`
6. Retry automático: max 3 tentativas com backoff
7. Progress tracking: lecture.processing_progress atualizado (0→25% durante transcrição)
8. Suporte a áudios de até 2 horas (chunked se necessário para API limits)
9. Error handling: se Whisper falha, job.error_message salvo e lecture.status = 'failed'
10. Após transcrição completa, cria automaticamente jobs para `summary`, `ebook`, `playbook`, `card`

## Dev Notes

**Whisper API call:**
```typescript
import OpenAI from 'openai'
const openai = new OpenAI({ apiKey: Deno.env.get('OPENAI_API_KEY') })

const transcription = await openai.audio.transcriptions.create({
  file: audioFile,        // File object
  model: 'whisper-1',
  language: 'pt',         // Portuguese hint
  response_format: 'text' // or 'verbose_json' for timestamps
})
```

**Whisper API limits:**
- Max file size: 25MB
- Supported formats: mp3, mp4, mpeg, mpga, m4a, wav, webm
- Para áudios > 25MB: split com ffmpeg ou programmaticamente

**Job cascade após transcrição:**
```typescript
// After successful transcription, create downstream jobs
const downstreamJobs = ['summary', 'ebook', 'playbook', 'card']
for (const type of downstreamJobs) {
  await supabase.from('processing_jobs').insert({
    lecture_id,
    type,
    status: 'queued'
  })
}
```
