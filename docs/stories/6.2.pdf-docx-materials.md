## Story

**As an** organizer,
**I want** to download lecture materials in PDF and editable DOCX format,
**so that** I can customize and redistribute the content.

## Acceptance Criteria

1. Edge Function `generate-materials` gera PDF e DOCX a partir do conteúdo processado (complementa Story 3.3 com templates profissionais e branding avançado)
2. PDF inclui: capa profissional, sumário, transcrição formatada, resumo, tópicos — via `pdf-lib` (Deno-native) usando `ScribiaPdfBuilder` de `@scribia/shared`
3. DOCX inclui mesmo conteúdo mas editável (Word-compatible) — via `npm:docx` (Deno-compatible)
4. Template profissional com branding estendido: header/footer, numeração de páginas, logo, cores corporativas (diferencia de 3.3 que gera versão básica)
5. No admin: botão "Gerar Materiais" por lecture com opções [PDF] [DOCX] [Ambos]
6. No portal do participante: botão "Download PDF" (se organizer habilitou)
7. Download tracking: incrementa download_count em lecture_access
8. Geração é assíncrona — mostra loading e notifica quando pronto

## Dev Notes

**DOCX generation com `docx` lib:**
```typescript
import { Document, Packer, Paragraph, HeadingLevel } from 'docx'

const doc = new Document({
  sections: [{
    children: [
      new Paragraph({ text: lecture.title, heading: HeadingLevel.TITLE }),
      new Paragraph({ text: `Palestrante: ${speaker.name}` }),
      new Paragraph({ text: lecture.summary }),
      // ... more content
    ]
  }]
})
const buffer = await Packer.toBuffer(doc)
```
